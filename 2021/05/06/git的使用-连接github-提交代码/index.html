git的使用-连接github-提交代码

github

首先，得有github账号，然后去创建一个新的项目仓库

git

  • 安装git
  • 新建目录，作为本地仓库，进入目录，右键打开 git base here
  • 执行 git init,告知该目录参考用git来管理
  • 执行 git clone SSH地址，克隆github项目代码
  • 修改项目代码，然后需要提交代码需要执行一下操作
    • 加入(指定文件)所有文件到暂存区
      git add .
      git add hello.txt
    • git commit -m '修改原因，相关说明信息'
    • 提交(根据情况选择)
      • git push 直接到远程默认仓库
      • git push orgin master push到名为orgin的远程仓库的指定分支master
  • 切换版本,然后再强行推送

分支查看与切换

  • 查看所有分支
  • 查看当前使用分支(结果列表中前面标*号的表示当前使用分支)
  • 切换分支

    多人协作

gitlab

  • 安装了git小乌龟之后，拉取gitlab项目
  • 新建目录，然后执行git clone（或者右键选择git 克隆）
  • 输入账号密码成功后即可克隆到代码
